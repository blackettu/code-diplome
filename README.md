# code-diplome

Код для части дипломного проекта. Репозиторий содержит набор Python-скриптов, связанных с обработкой изображений, детекцией объектов (YOLO) и анализом данных о сеянцах/растениях в контейнерах.

## Содержимое

- `learning_yolo.py` – обучение модели YOLO.
- `many_seedlings.py` – обработка/анализ множества сеянцев.
- `matrix_for_container.py` – работа с матрицей расположения растений в контейнере.
- `model_info.py` – вывод информации о модели.
- `filter.py` – фильтрация данных или изображений.
